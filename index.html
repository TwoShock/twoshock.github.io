Crime analysis and NLP

During the summer of my 3rd year I was interning with Prof. Bocar Ba on a project related to analyzing crime data. During this internship I had a variety of tasks to complete.

My main task revolved around curating data from various websites and then performing topic modeling(LDA) on the text data to extract the relevant topics in the text. I then performed unsupervised sentiment analysis on the topics to be able to graph the variation in sentiment by topic over time.

Finally, I was tasked to refactor a 2 year old custom crime preprocessing library. I updated the library to the latest python versions and fixed all errors along the way. I then modularized the code in order to make the package pip friendly and wrote a plethora of test cases.
